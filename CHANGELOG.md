## changelog 0.3.1.1 > 0.3.2

unified mango menus under a single CLI:

mangoconf view to view configurations

mangoconf edit to use the editor

both use an FZF tui to pick and preview config files. 

also added mangoconf help to show help.

lastly, code was altered to make it so that save+quit or escaping out of a specific config file brings you back to the main menu, quitting out of that with esc or ctrl+c will close the program.

this makes it more viable for use from the console.
