# Overview

SDG-MANGO-HELPERS provides a collection of small tools that make it easier to interact with and customize your mangoWM desktop. Each tool is launched from the terminal and presents interactive menus for common tasks.

| Tool | What it does |
|------|--------------|
| `dmsbars` | Switch between different bar styles (classic, Windows-like, mac-like, and more) |
| `manlayout` | Change how windows are arranged on screen |
| `mangoconf` | Browse and edit configuration files (`mangoconf edit`) or view line-by-line explanations (`mangoconf view`) |
| `swapmarked` | Exchange all windows between two workspaces |
| `sdgrgb` | Sync keyboard lighting with theme changes |
| `window-grouping` | Group and ungroup windows on a monitor |
| `layoutbutton` | Show the current layout and run per-layout actions |
