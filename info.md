Package Name: sdg-mango-helpers
Descriptive Name: SDG Mango Helper Tools
Source: https://github.com/SDG-Den/SDG-MANGO-HELPERS
Maintainer: SDGDen
Version:0.3.3

Dependencies: 
mangowm, dms, fzf, bat, jq, micro

Description: 
Collection of Bash shell scripts providing convenient TUI tools for configuring and interacting with mangoWM and DankMaterial Shell. Commands: dmsbars (bar preset picker), manlayout (layout switcher), mangoconf (config file browser/editor, with `mangoconf edit` and `mangoconf view` subcommands). Also provides workspace swapping and layout indicator scripts.
