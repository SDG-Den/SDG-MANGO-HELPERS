# Config Overview

MangoWM configuration browser and checker. Lets you browse config files
line-by-line with explanations for each setting.

## Commands

| Command | Keybind | Purpose |
|---------|---------|---------|
| `mangoconf-viewer` | SUPER+ALT+M | Interactive config line browser |
| `mangoconf-editor` | SUPER+M | Config file editor (opens in micro) |

## Scripts

| Script | Purpose |
|--------|---------|
| `menu.sh` | Main viewer TUI — select config, browse lines with details |
| `browser.sh` | Open documentation for a setting in Firefox |
| `checker.sh` | Validate config option names against reference list |
| `options.list` | Reference list of all config options with descriptions |
